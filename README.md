# Yandex Music Клон

Клон яндекс музыки без реализации функционала подкастов.

[MySQL Модели](https://drawsql.app/teams/kematins-team/diagrams/ym-clone)

### Технологии

**Backend:**
- Language
    - Python v. 3.10
- API
    - FastAPI
    - Uvicorn
- DB
    - MySQL
    - Redis
    - Elastic
- CI/CD
    - Flake8
    - GitHub Actions 
- Security
    - Password hashing
    - JWT tokens auth
- Tests
    - Pytest
    - Coverage
- Additions
    - Poetry as Package manager
    - Loguru for Logs

**Frontend:**
- Vite
- React
